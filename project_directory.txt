High-Level Architecture (v1.0)
              ┌────────────┐   1-min pull   ┌───────────────┐
              │  scheduler │───────────────▶│  Alpaca API   │
              └────────────┘                └───────────────┘
                    │
                    ▼
     ┌──────────────────────────────────────────────────┐
     │              data_ingest.py                     │

     │  · run LSTM + ARIMA + XGB ➜  predictions        │
     └──────────────────────────────────────────────────┘
                    │  (psycopg2 / SQLAlchemy)
                    ▼
         ┌────────────────────────────┐
         │     PostgreSQL (Docker)    │
         │  • market_data             │
         │  • predictions             │
         │  • sql_views (optional)    │
         └────────────────────────────┘
                    │
                    ▼
        ┌───────────────────────────────┐
        │      Streamlit dashboard      │
        │  • Ticker selector            │
